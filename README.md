## Quick Start

### 1. Using Apify Console (No Code)

1. Click any **"Try it"** link above
2. Configure your input (URLs, search terms, etc.)
3. Click **"Start"**
4. Download results as JSON, CSV, or Excel

### 2. Using API

```bash
# Run any scraper via API
curl -X POST "https://api.apify.com/v2/acts/intelligent_yaffle~amazon-product-scraper/runs?token=YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"startUrls": [{"url": "https://amazon.com/dp/B09V3KXJPB"}]}'
```

## API Integration

### Python

```python
from apify_client import ApifyClient

client = ApifyClient('YOUR_API_TOKEN')

# Run Amazon scraper
run = client.actor('intelligent_yaffle/amazon-product-scraper').call(
    run_input={'startUrls': [{'url': 'https://amazon.com/dp/B09V3KXJPB'}]}
)

# Get results
for item in client.dataset(run['defaultDatasetId']).iterate_items():
    print(item['title'], item['price'])
```

### Node.js

```javascript
import { ApifyClient } from 'apify-client';

const client = new ApifyClient({ token: 'YOUR_API_TOKEN' });

// Run Amazon scraper
const run = await client.actor('intelligent_yaffle/amazon-product-scraper').call({
    startUrls: [{ url: 'https://amazon.com/dp/B09V3KXJPB' }]
});

// Get results
const { items } = await client.dataset(run.defaultDatasetId).listItems();
console.log(items);
```
